##########################################################################################

This battery monitor daemon is designed to check battary status.

When pupper's battery overdischarges, it makes pupper suspend and then switch off servo power.

### run install.sh to deploy this daemon on you pupper

### use cmake and make command to rebuild it.(eg: cmake . ; make ), maybe you need install cmake tools fist.(sudo apt-get install cmake)

###

Source code and binary link : https://github.com/mangdangroboticsclub/QuadrupedRobot.git

By the way : i2cbusses.c and smbus.c is refer to https://i2c.wiki.kernel.org/index.php/I2C_Tools
